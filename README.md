**Project Title: Academic-Performance-Insights**
A data-driven analysis and visualization of student performance to uncover key academic insights.

**Overview:**
This project analyzes student performance data using Python and Excel to understand patterns in academic results. It provides a clean dashboard to visualize performance trends, subject comparisons, and improvement areas.

**Objectives:**
- Analyze raw student performance data
- Identify key factors affecting results
- Visualize overall and subject-wise performance.
  
**Project Files Description:**
1. Student_Performance_Raw_Data.xlsx	
- Contains original unprocessed student performance data (marks, attendance, subjects, etc.)
  
2. Student_Performance_DashBoard.xlsx
- Cleaned and processed Excel dashboard with charts, pivot tables, and visuals showing performance trends.

3. dashboard.png
- Screenshot showing the final dashboard view and key visuals

**Tools and Technologies:**
- Microsoft Excel.
- Pivot tables.
- Charts.
- Conditional formatting.

**Steps to Reproduce:**
- Download the repository.
- Open Student_Performance_Raw_Data.xlsx in Excel.
- Open Student_Performance_DashBoard.xlsx to view the dashboard.
